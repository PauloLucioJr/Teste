Como criar uma Lista de Desrição com html

Vamos usar as seguintes informações abaixo, como Exemplo:

Html5: Linguagem de marcação de contéudo

Css3: Linguagem de estilizaão de contéudo

Javascript: Linguagem de programação utilizada na criação de eentos nas páginas

Php: Linguagem de programação que roda em servidor utilização para geração de contéudo dinâmico

Mysql: "Linguagem" construída com base em SQL utilizada para gerenciamento de banco de dados
