Tag de Imagem (img)

A tag ela não possui fechamento assim como as outros acima. Ela abre e fecha nela mesma.

Você deve inserir a imagem da seguinte forma:

Exemplo

Src =source significa - origem

Imagem/DBZ2.JPG = Onde foi buscada a imagem, no caso, na pasta Imagem/nome do arquivo. extensão no caso JPG

“ “ = abre e fecha aspas para o endereço da imagem

/> = fecha tag da imagem

Outro exemplo

Src =source significa - origem

“ “ = abre e fecha aspas para o endereço da imagem

Width = tamanho da imagem

Height = Largura da imagem

Alt = Usado geralmente para pessoas segas, pois, quando chega nessa imagem é falado o que foi escrito entre as aspas.

Title = Usado geralmente pra quando a pessoa passa o mouse sobre a imagem, aparece a frase que você colocou usando esta tag.

/> = fecha tag da imagem
